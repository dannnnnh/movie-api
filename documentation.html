Movie API

Welcome to the Movie API! This API allows you to search for movies and get information about them, including their title, year of release, director, and actors.

Movie Club API Documentation

GET /users

Returns a list of all users in the system.

  • Endpoint URL: /users
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: An array of user objects. Example:
    •         [{"id": "ab12cd34",
		"name": "Alex",
		"favoriteMovies": ["The Shawshank Redemption", "The Godfather"]
          },
          {
            "id": "ef56gh78",
            "name": "Bob",
            "favoriteMovies": ["The Dark Knight", "The Matrix"]
          }
        ]

POST /users

Creates a new user in the system.

  • Endpoint URL: /users
  • Method: POST
  • Query Parameters: None
  • Request Body: A JSON object containing the new user's name. Example: 
    •         {
          "name": "Charlie"
        }
  • Response: A JSON object containing the newly created user's ID and name. Example: 
    •         {
          "id": "ij90kl12",
          "name": "Charlie"
        }

GET /movies

Returns a list of all movies in the system.

  • Endpoint URL: /movies
  • Method: GET
  • Query Parameters: None
  • Request Body: None
  • Response: An array of movie objects. Example:
    •         [          {"id": "001","title": "The Shawshank Redemption",
		"genre": {"name": "Drama"},"director": {"name": "Frank Darabont"}},
		 ]

FIND MOVIE BY TITLE

Returns the movie object that matches the provided title

  • Endpoint URL: /movies/title/:title
  • Method Type: GET
  • Query Parameters: title (string)
  • Body of the Request: N/A
  • Example: /movies/title/The Godfather
  • Response Data: A JSON object representing the movie

FIND MOVIE BY GENRE

Returns a list of all movies that belong to the provided genre

  • Endpoint URL: /movies/genre/:genreName
  • Method Type: GET
  • Query Parameters: genreName (string)
  • Body of the Request: N/A
  • Example: /movies/genre/Drama
  • Response Data: An array of JSON objects representing movies that belong to the specified genre

FIND DIRECTOR

Returns the director object that matches the provided director name

  • Endpoint URL: /movies/directors/:directorName
  • Method Type: GET
  • Query Parameters: directorName (string)
  • Body of the Request: N/A
  • Example: /movies/directors/Quentin Tarantino
  • Response Data: A JSON object representing the director

UPDATE USERNAME

Updates the name of the user with the provided ID

  • Endpoint URL: /users/:id
  • Method Type: PUT
  • Query Parameters: id (string)
  • Body of the Request: JSON object representing the updated user
    Example: {"name": "John Doe"}
  • Example: /users/1234
  • Response Data: A JSON object representing the updated user
    Example: {"id": "1234", "name": "John Doe"}

GET /movies?genre={genre}

Get all movies with a certain genre.

  • Endpoint URL: /movies?genre={genre}
  • Method: GET
  • Query Parameters: genre (string)
  • Request Body: None

GET /movies?genre={genre}&director={director}

Get all movies with a certain genre AND a certain director.

  • Endpoint URL: /movies?genre={genre}&director={director}
  • Method: GET
  • Query Parameters: genre (string), director (string)
  • Request Body: None

PUT /movies/{movieId}

Update the description of a particular movie.

  • Endpoint URL: /movies/{movieId}
  • Method: PUT
  • Query Parameters: movieId (string)
  • Request Body: A JSON object containing the new movie description. Example: 
    {
  "description": "New movie description"
}

PUT /movies/director/{directorId}

Update the bio of a certain director for multiple movies.

  • Endpoint URL: /movies/director/{directorId}
  • Method: PUT
  • Query Parameters: directorId (string)
  • Request Body: A JSON object containing the new director bio. Example: 
    {
  "bio": "New director bio"
}

PUT /users/{userId}/favorites

Add a movie to a user's list of favorites.

  • Endpoint URL: /users/{userId}/favorites
  • Method: PUT
  • Query Parameters: userId (string)
  • Request Body: A JSON object containing the movieId. Example: 
    {
  "movieId": "{movieId}"
}
