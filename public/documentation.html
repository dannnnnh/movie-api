Movie API Documentation

Endpoints

  • POST /login

    Authenticates a user and logs them in. Requires a JSON object containing the user's username and password. Needed to access other relevant endpoints.

  • GET /movies

    Returns a list of all movies.

  • GET /movies/:Title

    Returns a movie by its title.

  • GET /movies/genre/:genreName

    Returns a movie by its genre.

  • GET /movies/directors/:directorName

    Returns a movie by its director.

  • GET /users

    Returns a list of all users.

  • GET /users/:Username

    Returns a user by their username.

  • POST /users

    Creates a new user. Requires a JSON object containing Username, Password, Email, and Birthday.

  • PUT /users/:Username

    Updates an existing user. Requires a JSON object containing the updated Username, Password, Email, and Birthday.

  • POST /users/:Username/movies/:MovieID

    Adds a movie to the user's list of favorite movies.

  • DELETE /users/:Username/movies/:MovieID

    Removes a movie from the user's list of favorite movies.

  • DELETE /users/:Username

    Deletes a user by their username.
